1. Giải thích các câu lệnh
  - git commit --amend: Thay đổi commit message. Lệnh này sẽ mở ra 1 text editor để sửa commit
    - Gõ i -> Vào chế độ insert
    - Gõ esc để thoát insert
    - Gõ ":wq" -> Write and quit
  - git commit --amend -m”<message>”: Thay đổi commit message luôn mà không cần mở text editor
  - git restore --staged <file>: Đưa file từ vùng staging về working directory
  - git reset HEAD~1: Đưa 1 commit vùng repository về working directory (uncommit)
